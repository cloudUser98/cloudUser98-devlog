¿Cual es el proposito de esta web page?

En palabras simples, es un pateo de juegos personal donde puedo liberarme y hacer una de las cosas que mas disfruto, divertirme programando y aprendiendo.

Se que hay maneras mas ordenadas de realizar practicas y estudiar temas de desarrollo como tener un repositorio para cada proyecto, pero siento que tenerlo todo almacenado en una pagina web hara la experiencia mas entretenida ya que mi objetivo es fucionar el desarrollo web con los temas que aprenda sobre diferentes tecnologias que no tienen que forzosamente ser acerca desarrollo web.

Tal vez se pregunten, ¿Que cosas estudiare y plasmare en esta pagina?... Pues la verdad hay varios temas que llaman mi atencion haciendo a un lado el principal que es el desarollo web. Temas como programacion de bajo nivel, manejamiento de memoria, comunicacion de sockets, procesamiento en paralelo, ciencia en computacion, son los que por el momento llaman mas mi atencion, aunque claro, se que la lista seguira creciendo jajaja.
